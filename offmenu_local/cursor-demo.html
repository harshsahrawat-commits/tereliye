This is a complete rebuild of the interactivity from the target website. It includes the smooth, interpolating dot cursor with the mix-blend-difference effect, expansion on hover, and buttery smooth page scrolling.

Hover me (Link)

The cursor detects the hover event and dynamically scales up using a cubic-bezier CSS transition, mimicking the exact feel of offmenu.design.

Smooth Scrolling

The scrolling movement is powered by Lenis, overriding the native browser scroll with a mathematical easing function. Notice how both the scroll and the custom cursor feel decoupled from standard rigid browser mechanics.
